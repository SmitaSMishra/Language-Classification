# Language-Detection
Self learning AI model that classifies if the language is Engish(en), Dutch(nl) or Italian(it).
For Training Model:
  Input Parameters: train
  (train indicating to train both Decision Tree and Adaboost
For Prdiction:
  Input Parameters: predict tree/stump file_name.txt/Evaluate
    parameter 2: 'tree' if we want to predict using decision tree model and 'stump' if we want to predict using Adaboost
    parameter 3: 'file_name.txt' is the name of the file that contains the sentences to be predicted or if you want to evaluate the accuracy the 'evaluate'
